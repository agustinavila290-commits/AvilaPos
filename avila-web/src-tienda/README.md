# Módulo Tienda (futuro)

Esta carpeta contiene placeholders para la futura tienda online.

- **Catalogo.jsx** — Listado de productos.
- **Producto.jsx** — Detalle de producto.
- **Carrito.jsx** — Carrito y checkout.

Cuando se implemente la tienda, estos componentes pueden moverse a `src/pages/` y conectarse a las rutas en `App.jsx` (por ejemplo `/productos`, `/productos/:id`, `/carrito`). Opcionalmente se puede añadir un backend o API externa para el catálogo y los pedidos.
