# 🌐 GUÍA: CREAR PÁGINA WEB EN WORDPRESS

**Para:** Casa de Repuestos  
**Objetivo:** Presencia online, catálogo, contacto

---

## 📋 ¿QUÉ ES WORDPRESS?

WordPress es un **sistema de gestión de contenidos (CMS)** que te permite crear sitios web sin saber programar.

**Ventajas:**
✅ Fácil de usar  
✅ No necesitas ser programador  
✅ Miles de plantillas (temas) gratis  
✅ Plugins para agregar funcionalidades  
✅ SEO-friendly (Google lo ama)  
✅ Responsive (se adapta a móvil)  

---

## 🎯 OPCIONES PARA CREAR TU SITIO

### **Opción 1: WordPress.com (Gratis/Pago - RECOMENDADO PARA EMPEZAR)**
**Más fácil, todo incluido**

✅ **Ventajas:**
- Hosting incluido (no pagas servidor)
- Instalación automática
- Mantenimiento automático
- Backups automáticos
- SSL/HTTPS gratis

❌ **Desventajas:**
- Plan gratis tiene publicidad
- URL será: `tucasa.wordpress.com` (no personalizada)
- Limitaciones en plugins
- Plan de pago desde $4 USD/mes para quitar publicidad

**¿Para quién?**
- Principiantes
- Presupuesto limitado
- Solo necesitas un sitio informativo

---

### **Opción 2: WordPress.org (Auto-hospedado)**
**Más flexible, requiere hosting propio**

✅ **Ventajas:**
- Control total
- Tu propio dominio (www.tucasa.com)
- Sin limitaciones
- Miles de plugins gratis

❌ **Desventajas:**
- Debes contratar hosting ($3-10 USD/mes)
- Debes instalar WordPress
- Mantenimiento manual
- Requiere conocimientos básicos

**¿Para quién?**
- Negocios serios
- Quieres control total
- Presupuesto para hosting

---

## 🚀 GUÍA PASO A PASO: WORDPRESS.COM (OPCIÓN FÁCIL)

### **PASO 1: Crear Cuenta**

1. Ve a: https://wordpress.com/es/
2. Clic en **"Empezar"**
3. Ingresa tu email
4. Crea contraseña
5. Confirma email

---

### **PASO 2: Elegir Tipo de Sitio**

WordPress preguntará: **"¿Qué tipo de sitio web quieres crear?"**

Responde:
- **Tipo:** "Negocio" o "Tienda"
- **Objetivo:** "Vender productos" o "Promocionar mi negocio"
- **Nombre del sitio:** "Casa de Repuestos de Motos"

---

### **PASO 3: Elegir Dominio**

WordPress te ofrecerá:
- **Gratis:** `casaderepuestos.wordpress.com`
- **De pago:** `casaderepuestos.com` ($15-20 USD/año)

**Recomendación inicial:** Usa el gratis para probar, luego compra dominio propio.

---

### **PASO 4: Elegir Plan**

| Plan | Precio | Características |
|------|--------|-----------------|
| **Gratis** | $0/mes | Con publicidad, dominio WordPress |
| **Personal** | $4/mes | Sin publicidad, soporte |
| **Premium** | $8/mes | Diseño avanzado, plugins limitados |
| **Business** | $25/mes | Plugins ilimitados, tienda online |

**Recomendación:** Empieza con **Gratis** o **Personal** ($4/mes).

---

### **PASO 5: Elegir Plantilla (Tema)**

WordPress te mostrará temas. Busca:
- **"Auto parts"** (repuestos de auto)
- **"Motorcycle"** (motos)
- **"Shop"** (tienda)

**Temas recomendados:**
1. **Astra** (gratis, rápido, versátil)
2. **OceanWP** (gratis, para tiendas)
3. **Neve** (gratis, moderno)
4. **Storefront** (gratis, ideal para WooCommerce)

**Consejo:** Elige uno con **"Mobile-friendly"** y **"Fast loading"**.

---

### **PASO 6: Personalizar tu Sitio**

WordPress tiene un **editor visual** (arrastra y suelta).

#### **6.1 Crear Páginas Principales:**

1. **Inicio (Home)**
   - Banner con logo y eslogan
   - "Repuestos de motos de calidad"
   - Botón: "Ver Catálogo"

2. **Nosotros (About)**
   - Historia de tu negocio
   - Misión y visión
   - Fotos del local

3. **Productos (Shop)**
   - Catálogo de repuestos
   - Con fotos y precios
   - Categorías (Motor, Frenos, etc.)

4. **Contacto (Contact)**
   - Formulario de contacto
   - Dirección, teléfono, email
   - Mapa de Google Maps
   - Horarios de atención

#### **6.2 Agregar Contenido:**

**Para agregar una página:**
1. Panel izquierdo > **"Páginas"** > **"Añadir nueva"**
2. Escribe el título
3. Usa bloques para agregar:
   - Texto
   - Imágenes
   - Botones
   - Formularios
   - Videos

**Para agregar productos:**
1. Instala plugin **WooCommerce** (gratis)
2. Ve a **"Productos"** > **"Añadir nuevo"**
3. Título: "Pistón Honda CG 150"
4. Descripción
5. Precio
6. Imagen
7. Categoría
8. Publicar

---

### **PASO 7: Configurar Menú**

1. Panel > **"Apariencia"** > **"Menús"**
2. Crear menú: "Menú Principal"
3. Agregar páginas:
   - Inicio
   - Nosotros
   - Productos
   - Contacto
4. Asignar a: "Menú principal"
5. Guardar

---

### **PASO 8: Plugins Esenciales (Gratis)**

| Plugin | Función |
|--------|---------|
| **WooCommerce** | Tienda online |
| **Contact Form 7** | Formularios de contacto |
| **Yoast SEO** | Optimización para Google |
| **Wordfence Security** | Seguridad |
| **WP Super Cache** | Velocidad |
| **Google Analytics** | Estadísticas de visitas |

**Instalar plugin:**
1. Panel > **"Plugins"** > **"Añadir nuevo"**
2. Buscar nombre
3. Clic **"Instalar ahora"**
4. Clic **"Activar"**

---

### **PASO 9: SEO Básico**

Para que Google te encuentre:

1. **Títulos claros:** "Casa de Repuestos de Motos - Calidad y Precio"
2. **Descripciones:** Agrega descripción a cada página
3. **Palabras clave:** "repuestos de motos", "pistones Honda", etc.
4. **URLs amigables:** `/productos/piston-honda-cg-150`
5. **Google My Business:** Registra tu negocio en Google Maps

---

### **PASO 10: Publicar**

1. Revisa todo
2. Clic en **"Publicar"**
3. Tu sitio estará en: `casaderepuestos.wordpress.com`

### **PASO 2: Comprar Dominio**

**Opciones:**
- `.com` (internacional) - $10-15 USD/año
- `.com.ar` (Argentina) - $200-500 ARS/año
- `.shop` (tienda) - $10-20 USD/año

**Proveedores de dominios:**
- NameCheap
- GoDaddy

---

### **PASO 3: Instalar WordPress**

La mayoría de hostings tienen **instalador automático**:

1. Entra al panel de tu hosting (cPanel)
2. Busca **"WordPress"** o **"Softaculous"**
3. Clic en **"Instalar ahora"**
4. Elige:
   - Dominio
   - Título del sitio
   - Usuario admin
   - Contraseña
5. Clic **"Instalar"**
6. Espera 1-2 minutos
7. ¡Listo! Tu WordPress está en: `www.tudominio.com/wp-admin`

## 🔗 INTEGRACIÓN CON TU SISTEMA ACTUAL

### **Opción 1: Links desde WordPress a tu Sistema**
- En tu sitio WordPress, botón **"ACCESO VENDEDORES"**
- Link a: `http://192.168.1.23:5173` (tu sistema actual)
- Solo accesible desde tu red local o VPN

### **Opción 2: API Pública (Avanzado)**
- Exponer una API REST desde tu sistema Django
- WordPress consulta stock/precios en tiempo real
- Requiere configuración de seguridad (CORS, autenticación)

### **Opción 3: Catálogo Estático**
- Exportas productos a Excel
- Los subes manualmente a WordPress
- Actualizas precios semanalmente

**Recomendación inicial:** Opción 3 (manual) hasta que el negocio crezca.
