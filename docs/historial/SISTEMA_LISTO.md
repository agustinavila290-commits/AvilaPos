## 🔧 SI HAY PROBLEMAS

### Si Dark Mode no funciona:
1. **Limpia caché del navegador**: Ctrl + Shift + Delete
2. **Recarga forzada**: Ctrl + Shift + R
3. **Mira la consola**: ¿Aparecen los logs?
4. **Inspecciona el HTML**: Presiona F12 → Elements → Busca `<html class="dark">`

### Si se ve muy grande todavía:
1. **Verifica el zoom**: Presiona Ctrl + 0 (debe estar en 100%)
2. **Recarga con caché limpia**: Ctrl + Shift + R
3. **Prueba en ventana incógnito**: Ctrl + Shift + N
